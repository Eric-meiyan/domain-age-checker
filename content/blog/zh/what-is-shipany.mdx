## ShipAny 是什么

ShipAny 是一个基于 NextJS 的 AI SaaS 开发框架，内置丰富的组件和 AI SDK，帮助你快速完成 AI SaaS 项目开发。

## ShipAny 框架包含哪些功能

核心功能

- [x] 一个精美的 Landing Page
- [x] 20+ 页面组件
- [x] 用户登录（谷歌登录 / Github 登录）
- [x] 数据统计（Google Analytics / OpenPanel）
- [x] i18n 多语言
- [x] 数据存储（基于 Supabase）
- [x] 支付（基于 Stripe）
- [x] 部署（支持 Vercel / Cloudflare）

管理后台

- [ ] 用户管理
- [ ] 用户积分管理
- [ ] 支付订单管理
- [ ] API Keys 管理

AI SDK

- [x] 可灵 AI 视频生成
- [ ] Dall-E 图片生成
- [ ] OpenAI Chat Completion
- [ ] OpenAI TTS

> 其他需求可以在 [Github Issues](https://github.com/shipanyai/shipany-template-one/issues) 提交，按优先级支持。

## 技术依赖

ShipAny 基于 NextJS 框架构建，用到了诸多流行的技术组件和云服务。

为确保你能充分使用 ShipAny 的完整功能，请自行熟悉以下技术/组件或云服务：

技术/组件

- [NextJS](https://nextjs.org/docs) 全栈开发框架
- [TailwindCSS](https://tailwindcss.com/) CSS 样式框架
- [Shadcn/UI](https://ui.shadcn.com/docs) UI 组件库
- [next-intl](https://next-intl.dev/) i18n 多语言
- [next-auth](https://authjs.dev/) 登录鉴权

云服务

- [Supabase](https://supabase.com/) 云数据库
- [Stripe](https://stripe.com/) 支付收款平台
- [Google Analytics](https://analytics.google.com/) 数据统计
- [OpenPanel](https://openpanel.dev) 数据统计

## 环境依赖

为确保你能顺利在本地运行和二次开发 ShipAny，请在你的电脑上安装必要的环境。

以下为 Mac 系统开发的环境配置建议：

1. 使用 [fnm](https://github.com/Schniz/fnm) 进行 NodeJS 版本管理

推荐使用以下版本的 NodeJS 和 npm：

```bash
$ node -v
v22.2.0

$ npm -v
10.7.0
```

2. 使用 [pnpm](https://pnpm.io/installation) 进行包管理

全局安装 pnpm：

```bash
$ npm install -g pnpm
```

推荐使用以下版本的 pnpm：

```bash
$ pnpm -v
9.15.0
```

3. 安装 git 并配置 [ssh key](https://docs.github.com/en/authentication/connecting-to-github-with-ssh)，确保能访问你的 Github 仓库

```bash
$ git --version
git version 2.39.3 (Apple Git-146)
```

> 其他操作系统，请自行搜索对应的环境配置方法。运行 ShipAny 代码之前，请确保你的电脑环境已经安装了 nodejs 和 pnpm。
